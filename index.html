hashsplit.js - demo

github.com/klml/hashsplit is a JS plugin to use values from URL fragment identifier (hash) #foo , #foo/bar or #foo=bar&city=springfield.

prefill forms through url

link to #name=bob&city=springfield&color=green or link to #name=alice&city=shelbyville&color=&priority=1.

Check a checkbox #hashsplit_checkbox to uncheck use #hashsplit_checkbox=false or #hashsplit_checkbox=

Or combine all: #name=carol&city=north%20haverbrook/color=yellow/hashsplit_checkbox&priority=3
